## About ShopQueue
这是一个kafka队列实战商城，kafka作为项目的基础架构，完善各个功能细节，日志存储，削锋解耦，异步处理，定时任务处理
所用到的技术
- Kafka
- Redis
- Elasticsearch
- Docker
- Jenkins
- MySQL
- Nginx
- PHP8.0
